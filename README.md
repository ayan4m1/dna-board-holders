## Printing

Recommended print settings:

| Setting      | Value  |
| ------------ | ------ |
| Perimeters   | 3+     |
| Layer Height | ~0.2mm |
| Infill       | 20%    |
| Supports     | Off    |
| Raft         | Off    |
| Brim         | Off    |
